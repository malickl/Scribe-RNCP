# Scribe

Assistant de réunion intelligent. Voir le document de contexte du projet pour l'architecture complète.

## Installation

1. `pip install -r requirements.txt`
2. Copier `.env.example` en `.env` et remplir les clés
3. Ajouter `credentials.json` (OAuth Google) à la racine
4. `python db/init_db.py` pour créer les tables
5. `python app.py`
